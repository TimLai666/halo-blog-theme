# NEURAL (Halo Theme)

一套「首頁索引 / 文章沉浸」的雙模式主題：  
- 首頁：深色、卡片、資訊密度，適合逛文章  
- 文章頁：亮底、極少 UI、排版主導，適合長文閱讀  
- 沒封面也好看（封面是可選的 ambient）

## 安裝
1. 把整個 `neural` 資料夾放到 Halo 的 `themes/` 下面（Halo 2.0+）
2. 到 Console → 外觀 → 主題 → 啟用 `NEURAL`
3. 若修改了 `settings.yaml` 或 `theme.yaml`，記得點「重載主題配置」

## 主題設定
Console → 外觀 → 主題 → 設定  
- 預設配色、強調色、是否襯線內文、閱讀進度條等

## 自訂文章模板
這個主題在 `theme.yaml` 提供一個自訂文章模板：`post_read.html`
文章 → 編輯 → 模板 → 可選「沉浸閱讀（亮底）」
